Help & Support

Welcome to the Steganography Studio help page. Follow the steps below to use our platform effectively.

1. Embedding Messages

  • Navigate to the "Embed" section from the menu.
  • Upload your host file (image, text).
  • Enter the message you want to hide.
  • Choose security options like encryption (optional).
  • Click the "Embed" button to hide the message in the file.
  • Download the steganographic file for sharing.

2. Extracting Messages

  • Go to the "Extract" section from the menu.
  • Upload the steganographic file containing the hidden message.
  • Provide the decryption key if applicable.
  • Click the "Extract" button to reveal the hidden message.

3. Editing Files

  • Use the "Editor" section to modify or preview your files.
  • Upload a file and adjust encoding parameters as needed.
  • Preview results before saving changes.

4. Troubleshooting

  • If the file fails to upload, ensure it meets the supported format.
  • Check if the decryption key is correct when extracting messages.
  • Contact support if you encounter persistent issues.
